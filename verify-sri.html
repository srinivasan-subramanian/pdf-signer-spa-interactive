What is SRI?

Subresource Integrity (SRI) ensures that resources haven't been tampered with by:

  • ✅ Generating cryptographic hashes of your assets
  • ✅ Adding integrity attributes to script/link tags
  • ✅ Browser verifies hash matches before executing
  • ✅ Blocks execution if hash doesn't match (prevents tampering)
